== Description ==

Das PS-Mediathek BoniPress-Addon ermöglicht es Webseitenbesitzern, ihren Benutzern Punkte für das Hochladen von Medien zu geben. Für verschiedene PS-Mediathek-Aktionen können unterschiedliche Punkte vergeben werden.

Derzeit unterstützte Aktionen sind:

*   Foto-Upload
*   Video Upload
*	Audio Upload
*	Dokument Upload
*	Foto Löschen
*	Video Löschen
*	Dokument Löschen
